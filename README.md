#flipjs

Flip things! Flip strings! Go mad! Test unicode support!

##Command Line Usage

###Installation
```bash
npm install -g flip
```
###Usage

```bash
$ flip A Table
(╯°□°）╯︵ ǝʃqɐ⊥ ∀
```
